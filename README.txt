version 1.0.0.0
----
[2018/11/12]
تعتمد لعبة كانسة الألغام على التفكير ملياً
ويمكنك الفوز بها وذلك بحسب مستوى ذكائك
--------
تمكنك اللعبة من اللعب  وحدك أو مع الحاسب أو اللعب مع عدة لاعبين (ثمانية على الأكثر)
تستطيع تغيير إعدادات اللعبة (عرض الرقعة - طول الرقعة - عدد الألغام في الرقعة - المدة الزمنية لكل لاعب - عدد الدروع القوية والضعيفة لكل لاعب)
By :Customize->custome game

تستطيع تغيير إعدادات اللعبة (أي نقاط الربح والخسارة مثلاً عند الضغط على اللغم كم سيخصم من نقاط اللعب )
By :Customize->custome points

تستطيع عرض نتائج الألعاب السابقة المنتهية من الزر 
By : ScoreBoard

تستطيع تحميل ألعاب منتهية لعرضها أو ألعاب غير منتهية لتكملتها من الزر
 By : Load Game->استعراض
By : Load Game ->استئناف لعبة مؤقتة (Load Quick)
By : Load Game->استئناف 

  تستطيع حفظ ألعاب غير منتهية من أجل تكملتها بعد وقت قصير ولكن دون الخروج من اللعبة (وتعتبر آخر عملية حفظ هي التي سيتم حفظها )
By : start-> QuickSave

تستطيع حفظ ألعاب غير منتهية من أجل تكملتها فيما بعد أو منتهية من أجل عرضها 
By :  start -> Save Game

قبل تشغيل اللعبة عليك أولاً أن تقوم بإضافة مجلد في القرص 
D:\\
وتقوم بتسميته 
-Store
فيصبح المسار
 D:\\Store
ثم تضيف ثلاث مجلدات إليه
1- Game_Complete
2- Not_Game_Complete
3- Score_Board
فتصبح مسارات المجلدات  على التوالي كالتالي
 D:\\Store\\Game_Complete
 D:\\Store\\Not_Game_Complete
 D:\\Store\\Score_Board
( يجب أن تكون أسماء المجلدات كالأسماء السابقة تماماً )
----------------------------------------------------------------------------------------

Run MinesWeeperGame.bat

Minesweeper_Game1\dist \
